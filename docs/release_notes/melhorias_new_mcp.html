MCP Server — Contatos

Servidor Model Context Protocol sobre a API REST de Contatos

MCP FastMCP FastAPI Python 3.11+ 4 Tools
ℹ️
Objetivo: Expor as operações de CRUD de contatos como tools MCP para que agentes externos possam interagir com a base de dados sem precisar conhecer os detalhes HTTP da API. O MCP atua como cliente HTTP da API FastAPI existente (http://localhost:8000), reutilizando toda a lógica de negócio, validação e auditoria já implementada.

⚙️ Abordagem

O servidor MCP é implementado em Python usando o SDK oficial FastMCP (pacote mcp[cli]). Ele faz chamadas httpx para a API REST local e retorna os resultados formatados como saída de tools. A autenticação JWT é obtida via POST /auth/login de forma lazy e renovada automaticamente em caso de 401.

🛠️ As 4 Tools MCP

1. listar_contatos

  • Endpoint: GET /contatos/?limit=1000
  • Input: nenhum parâmetro
  • Output: lista de dicionários com id, nome, email, telefone, empresa, observacoes, criado_em

2. criar_contato

  • Endpoint: POST /contatos/
  • Obrigatórios: nome: str, email: str
  • Opcionais: telefone: str (formato (XX) XXXXX-XXXX), empresa: str, observacoes: str
  • Output: contato criado com id
  • Erros: 400 se email duplicado · 422 se campos inválidos

3. atualizar_contato

  • Endpoint: PATCH /contatos/{id} (atualização parcial)
  • Obrigatório: id: int
  • Pelo menos 1: nome, email, telefone, empresa, observacoes
  • Output: contato atualizado
  • Erros: 404 se não encontrado · 400 se email duplicado

4. excluir_contato

  • Endpoint: DELETE /contatos/{id}
  • Input: id: int
  • Output: mensagem de confirmação (soft delete — marca deletado_em, não remove o registro)
  • Erros: 404 se não encontrado

💻 Código — mcp/server.py

"""MCP Server — Contatos (wraps FastAPI REST API)."""
import os
from typing import Optional

import httpx
from dotenv import load_dotenv
from mcp.server.fastmcp import FastMCP

load_dotenv()

API_URL = os.getenv("API_URL", "http://localhost:8000")
_token: Optional[str] = None

mcp = FastMCP("contatos-mcp")


def _login() -> str:
    resp = httpx.post(f"{API_URL}/auth/login", json={
        "email": os.getenv("MCP_USERNAME"),
        "senha": os.getenv("MCP_PASSWORD"),
    })
    resp.raise_for_status()
    return resp.json()["access_token"]


def _headers() -> dict:
    global _token
    if not _token:
        _token = _login()
    return {"Authorization": f"Bearer {_token}"}


def _request(method: str, path: str, **kwargs) -> httpx.Response:
    # Renova token automaticamente em caso de 401
    global _token
    resp = httpx.request(method, f"{API_URL}{path}", headers=_headers(), **kwargs)
    if resp.status_code == 401:
        _token = _login()
        resp = httpx.request(method, f"{API_URL}{path}", headers=_headers(), **kwargs)
    return resp


@mcp.tool()
def listar_contatos() -> list[dict]:
    """Lista todos os contatos cadastrados na base."""
    resp = _request("GET", "/contatos/", params={"limit": 1000})
    resp.raise_for_status()
    return resp.json()["items"]


@mcp.tool()
def criar_contato(
    nome: str,
    email: str,
    telefone: Optional[str] = None,
    empresa: Optional[str] = None,
    observacoes: Optional[str] = None,
) -> dict:
    """Cria um novo contato. nome e email são obrigatórios."""
    payload = {"nome": nome, "email": email}
    if telefone: payload["telefone"] = telefone
    if empresa: payload["empresa"] = empresa
    if observacoes: payload["observacoes"] = observacoes
    resp = _request("POST", "/contatos/", json=payload)
    resp.raise_for_status()
    return resp.json()


@mcp.tool()
def atualizar_contato(
    id: int,
    nome: Optional[str] = None,
    email: Optional[str] = None,
    telefone: Optional[str] = None,
    empresa: Optional[str] = None,
    observacoes: Optional[str] = None,
) -> dict:
    """Atualiza parcialmente um contato. Informe apenas os campos a alterar."""
    payload = {k: v for k, v in {
        "nome": nome, "email": email, "telefone": telefone,
        "empresa": empresa, "observacoes": observacoes,
    }.items() if v is not None}
    resp = _request("PATCH", f"/contatos/{id}", json=payload)
    resp.raise_for_status()
    return resp.json()


@mcp.tool()
def excluir_contato(id: int) -> str:
    """Exclui (soft delete) um contato pelo ID."""
    resp = _request("DELETE", f"/contatos/{id}")
    resp.raise_for_status()
    return f"Contato {id} excluído com sucesso."


if __name__ == "__main__":
    mcp.run()
